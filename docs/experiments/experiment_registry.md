# PMDD Experiment Registry

This document serves as the formal audit trail for all benchmark experiments executed on the PMDD platform. Researchers must log their batch runs here to ensure thesis traceability.

## Log Structure
- **Date**: YYYY-MM-DD
- **Experiment ID**: e.g., `exp_002`
- **Corpus**: The specific dataset used.
- **Frameworks**: Which Agent frameworks were active?
- **Governance Overrides**: Any manual force-routing applied?
- **Result Summary**: Key findings (ECE, Drift Index, etc.)
- **Status**: (Draft / Published / Discarded)
