## 1. Production Frontend Deployment

The React frontend has been optimized and pre-configured for instant deployment on Netlify or Vercel.

**Netlify Configuration (`netlify.toml`):**
- **Build Command**: `npm run build`
- **Publish Directory**: `dist`
- **Routing**: `/*` rewrites to `/index.html` (Crucial for React Router stability)
- **Environment**: Node 20.

**Deployment Action Required:**
Because this is a locally hosted Git repository on your machine, to finalize the live link, you simply need to push this repository to GitHub, and connect the repository to your Netlify account. Netlify will automatically detect the `netlify.toml` file and deploy the production build.

*Your Live URL will be generated by Netlify upon the first push (e.g., `https://pmdd-research.netlify.app`).*

## 2. API Backend Deployment

The frontend must communicate with your Python backend (`api/server.py`). 
To deploy the backend:
1. Push the repository to a Python-friendly hosting service (e.g., Render, Railway, or Heroku).
2. The start command for the backend is: `uvicorn api.server:app --host 0.0.0.0 --port $PORT`
3. Ensure your `OPENAI_API_KEY` is securely added to the host's environment variables.
4. Once the backend is deployed, update your frontend's API base URL (in `MathematicalEngine.jsx` and `LongitudinalDashboard.jsx`) to point to your new live backend URL instead of `http://localhost:8000`.
